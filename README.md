# Setup

Use Node.js v20.

```
git clone https://github.com/Dirain1700/rankbot.git

cd rankbot

npm install

cp config/config-example.ts config/config.ts

cp src/detect-zh/setup/config-example.ts src/detect-zh/setup/config.ts
```

Edit config.ts, run `npm start`!
